## How to install



### From PyPI with pip

Install `types-aioboto3` for `EC2` service.

```bash
# install with aioboto3 type annotations
python -m pip install 'types-aioboto3[ec2]'


# Lite version does not provide session.client/resource overloads
# it is more RAM-friendly, but requires explicit type annotations
python -m pip install 'types-aioboto3-lite[ec2]'


# standalone installation
python -m pip install types-aiobotocore-ec2
```



## How to uninstall

```bash
python -m pip uninstall -y types-aiobotocore-ec2
```
